# Usage in Twig Instructions
<br>
### Use `breithbarbot_cropper_asset` function
```twig
{{ breithbarbot_cropper_asset(your_entity.your_field_name) }}
```
<br>
### Or use `asset` function
```twig
{{ asset(your_entity.your_field_name.path) }}
```
